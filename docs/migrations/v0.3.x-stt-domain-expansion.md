# STT Domain Model Expansion - Migration Guide (v0.3.x)

This guide covers the additive changes introduced in the STT domain model expansion (Issue #894). All changes are non-breaking: existing constructors and APIs continue to work unchanged.

## Summary of Changes

- **Non-breaking:** Enhanced `Transcription` with new helper methods
- **Non-breaking:** Added typed validation factories for `STTOptions` and `WordTimestamp`
- **Non-breaking:** Providers now populate `processingTimeMs` metadata
- **Non-breaking:** Richer `STTError` hierarchy with `retryable` and `userFriendly`

## Key Additions

### 1. New Typed Validation Factories

`STTOptions.validate()` returns a `Result` instead of throwing on invalid input:

```scala
STTOptions.validate(
  language = Some("en-US"),
  confidenceThreshold = 0.85
) match {
  case Right(opts)  => provider.transcribe(audio, opts)
  case Left(error)  => logger.error(error.userFriendly)
}

// Direct construction still works for known-valid values
val opts = STTOptions(
  language = Some("en-US"),
  confidenceThreshold = 0.85
)
```

`WordTimestamp.validate()` works the same way:

```scala
WordTimestamp.validate(
  word = "hello",
  startSec = 0.0,
  endSec = 0.5,
  confidence = Some(0.95)
) match {
  case Right(ts)   => /* use timestamp */
  case Left(error) => /* handle */
}
```

### 2. Enhanced Transcription Helper Methods

```scala
val trans: Transcription = ???

// Confidence analytics (over timestamped words)
trans.averageConfidence      // Option[Double]
trans.minConfidence          // Option[Double]
trans.maxConfidence          // Option[Double]

// Word and speaker analysis
trans.wordCount              // Int
trans.wordsBySpeaker(1)      // List[String]
trans.speakerSegments        // Map[Int, List[(Double, Double)]]

// Quality checks
trans.meetsQualityThreshold(minConfidence = 0.8, minWords = 3)  // Boolean

// Filtering
trans.filterByConfidence(0.5)  // Returns words with confidence >= 0.5;
                               // words without a confidence score are excluded.

// Duration
trans.totalDuration          // Option[Double] â€” max endSec across timestamps
                             // (handles unsorted timestamp lists correctly)
```

### 3. Provider Improvements

Both `VoskSpeechToText` and `WhisperSpeechToText` now populate additional metadata when available:

```scala
val transcription = provider.transcribe(audio, options)

transcription.processingTimeMs                  // Some(245)
transcription.timestamps.map(_.confidence)      // word-level confidence when provided
```

## Backward Compatibility Matrix

| Component        | Backward Compatible | Notes                                                                                       |
|------------------|---------------------|---------------------------------------------------------------------------------------------|
| `STTOptions`     | Yes                 | Direct construction unchanged; `validate()` adds Result-based validation                     |
| `WordTimestamp`  | Yes                 | Direct construction unchanged; `validate()` added                                            |
| `Transcription`  | Yes                 | New helper methods are additive                                                              |
| `STTError`       | Yes                 | Existing variants unchanged; richer fields (`retryable`, `userFriendly`, `context`) added    |
| Provider APIs    | Yes                 | Providers fill in optional metadata fields where supported; no signature changes             |

## Adoption Steps

### Switch from exceptions to Result for validation

```scala
// Before â€” throws IllegalArgumentException
val opts = STTOptions(confidenceThreshold = userInput)

// After â€” Result-based, no exceptions
STTOptions.validate(confidenceThreshold = userInput) match {
  case Right(opts)            => transcribe(audio, opts)
  case Left(error: STTError)  => logger.error(error.userFriendly)
}
```

### Use the new analytics helpers

```scala
if (trans.meetsQualityThreshold(minConfidence = 0.8, minWords = 5))
  processTranscription(trans)
else
  logFailure(trans.averageConfidence, trans.wordCount)
```
